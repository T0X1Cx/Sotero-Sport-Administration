Bienvenidos a Sotero Sport - Admin

  • Aplicación Web creada con Angular y Angular Material para la gestión de administrador del proyecto sotero-sport.

Uso

  • Para navegar por la web tan solo tenemos que irnos al sidebar e ir navegando por las diferentes páginas.

  • Podemos visualizar todos los registros de la base de datos, ver su id, email, role...

  • Editar los registros existentes: Si pulsamos sobre el botón de editar, se nos mostrará un formulario con todos los campos del registro.

  • Crear nuevos registros: Pulsamos sobre nuevo registro (usuario, item, lista...), se nos mostrará el formulario a rellenar para crear el nuevo registro.

  • Eliminar registros existentes: Si pulsamos sobre el botón borrar, se nos mostrará un diálogo de confirmación el cual debemos aceptar para borrar o podemos cancelarlo.
